User

The user extends the DataObject. Here you can find an example on how to implement data objects. You can override the default set/get methods, to match your expected behaviour.

Note, that there is no validation of input. It's a simple interface. Except for the id, as it is required there. Please use strings for IDs to avoid integer overflow issues (if ever).
